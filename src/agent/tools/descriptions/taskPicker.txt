Use this tool to pick the next task from the implementation plan.

WHEN TO USE:
- At the start of a new Ralph iteration
- When you need to select which task to work on next

BEHAVIOR:
- Reads the implementation plan from .fraude/
- Selects the first pending task (by priority)
- Marks the task as "in-progress"
- Returns task details for execution

This tool is part of the Ralph prompting workflow for iterative agent execution.
