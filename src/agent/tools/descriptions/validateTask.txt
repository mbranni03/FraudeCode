Use this tool to validate that a task has been completed successfully.

WHEN TO USE:
- After completing work on a task
- To verify acceptance criteria are met

INPUTS:
- taskId: The ID of the task to validate
- checks: Array of validation checks to perform

VALIDATION TYPES:
- fileExists: Check if a file exists at the given path
- contentContains: Check if a file contains specific text
- commandSucceeds: Run a command and check exit code is 0

If all checks pass, the task is marked as complete.
If any check fails, the task remains in-progress with blockers noted.

This tool is part of the Ralph prompting workflow for iterative agent execution.
